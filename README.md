## Overview
This project explores rough volatility dynamics based on the "Volatility is Rough" framework, using realized volatility data.

## Objective
- Analyze the roughness of volatility time series
- Study log-volatility behavior and scaling properties
- Replicate key empirical findings from the literature

## Methodology
- Use realized volatility data (Oxford-Man Institute dataset)
- Transform volatility series (log, square root)
- Empirical analysis of scaling behavior and roughness
- Visualization and statistical diagnostics

## Key Insights
- Volatility exhibits rough behavior, inconsistent with classical Brownian-based models
- Log-volatility dynamics show persistent irregularity across time scales
- Empirical properties support the use of rough volatility models for short-term dynamics
- Data quality and preprocessing significantly impact observed roughness

## Implementation
- Python
- Time series analysis
- Empirical/statistical exploration

## Files
- `project_v_final.ipynb`: main analysis and replication
- `rough_volatility_report.pdf`: detailed academic report (theoretical background and results)
